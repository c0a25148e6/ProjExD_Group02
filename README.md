# DOOM風FPSゲーム

MiniPy3DR(pygame製の3Dレンダラ)のFPSツールキット `minipy3dr.fps` 
を使ったDOOM風FPSゲーム

## 実行環境の必要条件

* Python >= 3.10
* pygame >= 2.5
* minipy3dr >= 0.5.0 (`pip install minipy3dr`)

## 実行方法

```
python main.py
```

## ゲームの概要
* 主人公キャラクター豪華豚（ゴウカトン）をマウス操作により丸焼きにするゲームで，．．．
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* 矢印キーで豪華豚を操作し，スペースキー押下によるバーナーで豪華豚を丸焼きにする
* 豪華豚が10秒以上バーナーにあぶられたら，ゲームオーバーとなる

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画

### 分担追加機能
* 効果音追加（担当：笹本）:銃声音などを付けた
* 背景音追加（担当：笹本）:後ろで流れるBGM追加した
* 色変更（担当：笹本）:床・壁の色を変更した

### メモ
* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある
